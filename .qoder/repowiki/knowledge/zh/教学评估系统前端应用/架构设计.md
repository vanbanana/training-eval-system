- **技术栈**: 采用 Vue 3 (Composition API) + TypeScript + Vite 构建，使用 Pinia 进行状态管理，Vue Router 处理路由。
- **UI 架构**: 基于 shadcn-vue (reka-ui) 和 Tailwind CSS v4 实现原子化组件设计，通过 `components.json` 配置路径别名与样式变量。
- **分层结构**: `src/views` 按角色（admin/teacher/student）划分业务视图；`src/components` 分为基础 UI (`ui`)、布局 (`layout`) 和业务组件 (`business`)；`src/composables` 封装逻辑复用（如 WebSocket、通知）。
- **网络层**: `src/api/client.ts` 配置 Axios 全局拦截器，实现 JWT Token 自动附加与 401 错误下的无感刷新机制。
