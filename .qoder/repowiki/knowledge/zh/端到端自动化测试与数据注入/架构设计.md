- **分层架构**：模块内部划分为核心测试引擎（`e2e_full_flow.py`, `e2e_comprehensive.py`）、数据注入工具（`seed_real_data.py`）及诊断脚本（`check_db*.py`）。
- **执行入口**：通过 `run_e2e.ps1` 编排 Go 后端编译、种子数据初始化及 Python 测试脚本的串行执行流程。
- **通信机制**：基于 `httpx`/`requests` 封装 `APIClient`，实现 JWT 自动携带、请求耗时记录及状态码断言。
- **报告生成**：内置 `Metrics` 和 `TestReport` 类，实时采集 API 成功率、解析覆盖率及 P95/P99 延迟等量化指标，并输出 JSON/PDF 格式的详细评估报告至 `e2e_test_output/`。
